# Letterboxd Rutracker Checker

Расширение для Firefox, которое показывает наличие фильмов на Rutracker прямо на сайте Letterboxd.

## Установка

### Быстрая установка (рекомендуется)

**[Установить расширение](https://addons.mozilla.org/firefox/downloads/file/4664210/6e18e07e08094975ac0e-1.0.xpi)** — нажмите и подтвердите установку в Firefox.

> Требуется Firefox 140 или новее

---

## Что делает расширение?

Когда вы просматриваете фильмы на сайте [Letterboxd.com](https://letterboxd.com), расширение автоматически проверяет, есть ли этот фильм на торрент-трекере Rutracker, и показывает результат в виде небольшого значка.

### Где работает:
- **Страница фильма** — значок появляется рядом с названием
- **Watchlist** — значки на каждом постере в вашем списке "хочу посмотреть"
- **Списки фильмов** — значки на постерах в любых списках

### Что означают цвета значка:

| Цвет | Значение |
|------|----------|
| **Зелёный** | Раздачи найдены (число = количество раздач) |
| **Красный** | Раздачи не найдены |
| **Оранжевый** | Нужно войти в аккаунт Rutracker |
| **Серый** | Ошибка проверки |

При нажатии на значок откроется страница поиска на Rutracker.

---

## Как пользоваться

1. **Войдите в аккаунт Rutracker** — откройте [rutracker.me](https://rutracker.me) и авторизуйтесь. Это нужно сделать один раз, браузер запомнит вашу сессию.

2. **Откройте Letterboxd** — перейдите на любую страницу фильма, например:
   - https://letterboxd.com/film/the-matrix/
   - Или ваш watchlist
   - Или любой список фильмов

3. **Смотрите на значки** — рядом с названием фильма (или на постерах) появятся значки с информацией о раздачах.

4. **Нажмите на значок** — откроется страница поиска Rutracker с этим фильмом.

## Возможные проблемы

### Все значки оранжевые / показывают "RT (войти)"

**Причина:** Вы не авторизованы на Rutracker.

**Решение:**
1. Откройте [rutracker.me](https://rutracker.me) в новой вкладке
2. Войдите в свой аккаунт
3. Обновите страницу Letterboxd

### Значки не появляются

**Причина:** Расширение не установлено или страница не обновлена.

**Решение:**
1. Проверьте, что расширение установлено: `about:addons`
2. Обновите страницу Letterboxd (F5)
